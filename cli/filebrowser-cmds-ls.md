# filebrowser cmds ls

## Synopsis

List all commands for each event.

```
filebrowser cmds ls [flags]
```

## Options

| Name | Shorthand | Usage |
|------|-----------|-------|
|event|e|event name, without 'before' or 'after'|
|help|h|help for ls|
