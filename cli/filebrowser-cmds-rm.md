# filebrowser cmds rm

## Synopsis

Removes a command from an event hooker.

```
filebrowser cmds rm [flags]
```

## Options

```
  -e, --event string   corresponding event
  -h, --help           help for rm
  -i, --index uint     command index
```

### Inherited

```
  -d, --database string   path to the database (default "./filebrowser.db")
```
